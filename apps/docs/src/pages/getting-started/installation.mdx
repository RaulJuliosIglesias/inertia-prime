# Installation

Inertia Prime has two main parts:

- `@inertia-prime/react` – React components and hooks.
- `inertia-prime/laravel` – Laravel integration helpers.

## 1. Install the React package

```bash
# npm
npm install @inertia-prime/react @inertiajs/react react react-dom

# pnpm
pnpm add @inertia-prime/react @inertiajs/react react react-dom
```

## 2. Install the Laravel package

```bash
composer require inertia-prime/laravel
```

Laravel will auto-discover the service provider. If you prefer manual registration, add it to `config/app.php`:

```php
'providers' => [
    // ...
    InertiaPrime\\InertiaPrimeServiceProvider::class,
],
```
