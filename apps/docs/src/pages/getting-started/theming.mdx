# Theming

Inertia Prime ships with **Tailwind-first** styling and a set of design tokens.

## Theme goals

- Sensible defaults out of the box.
- Easy to adapt to your existing Tailwind config.
- Optional headless usage for full visual control.

## Headless vs styled

- **Styled components**: ready-to-use components with Tailwind classes.
- **Headless components/hooks**: behavior only, you bring the HTML & CSS.
